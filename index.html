Instructions:
  1. Select a marking mode from the options on the right.
  2. Click on a cell in the grid to mark it according to the selected mode.
  3. Enter the number of
  4. Click "Calculate" to run the solver.
  5. The simulation will highlight the cell with the highest probability of containing a ship.
  6. Adjust the marking mode and iterate as necessary to find all the ships.
Notes:
  1. When marking a ship as sunk, mark all the squares containing the ship with the same index i.e. S1. The solver works out which ships of which lengths have been sunk with this information
  2. The 'Diagonal Skew' option weights alternating diagonals with higher probabiilty - this helps to find the 2 length ship. Consider turning it off when the 2 length ship has been found.
  3. You can change number of iterations - the simulation will be more accurate at the cost of computing time. For constrained boards, the simulation can take a long time, 10,000 iterations is a good middleground.
